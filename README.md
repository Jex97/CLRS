* Content: c++翻译的书本中的伪代码，正文内容;
* Exercise: 每一小节后的练习题;
* Problem: 每一章后的思考题;
* Algorithm: 每一章的算法汇总，前三者的汇总，命名以算法名字重命名cpp文件;
